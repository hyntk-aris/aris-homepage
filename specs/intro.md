# Getting Started

Welcome to the ARIS Homepage Design System documentation.

This spec kit documents all UI components, sections, and design tokens used throughout the ARIS Vietnam website.

## Quick Links

- **Components**: Reusable UI elements (Button, Card, Grid, etc.)
- **Sections**: Full-page components (Hero, Services, Features, etc.)
- **Styles**: Design tokens (Colors, Typography, Spacing)

## For Designers

Use this documentation to understand component behavior and usage patterns. Keep designs aligned with these specs to ensure consistency across the product.

## For Developers

Refer to the component specs for implementation details, props, and usage examples. Follow the established patterns to maintain code consistency.

## Contributing

When adding new components or features:
1. Create a spec document in the appropriate folder
2. Include usage examples and props documentation
3. Update the spec-kit.config.js navigation if needed
4. Commit and push changes
